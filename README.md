# 📊 Amazon Product Data Analysis using Python

## 🔍 Overview

This project analyzes Amazon product data to understand pricing, discounts, and customer ratings.

## 🛠 Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn

## 📊 Analysis Performed

* Price analysis
* Category distribution
* Rating analysis
* Correlation heatmap

## 📈 Key Insights

* Most products fall within a common price range
* Highly rated products indicate customer satisfaction
* Discounts impact pricing significantly

* sales_analysis.ipynb
* sales_dataset.csv
